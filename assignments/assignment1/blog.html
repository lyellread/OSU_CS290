Assignment 1 was reasonably simple to complete. Even having never written HTML before, I picked it up pretty fast and was able to put together a simple set of HTML only webpages in about an hour. As far as learnings from this assignment, I learned everything I did - that is to say I did not have any previous knowledge of HTML. I used a video guide to HTML to aid in my learning. I already knew how to use GitHub.
